# Naming Conventions

## Repository

Use a broad, descriptive repository name.

Recommended:

- `windows-batch-toolkit`

## Script Names

Use:

- lowercase
- dash-separated
- action-oriented names

Examples:

- `flush-dns.bat`
- `show-ip-config.bat`
- `backup-folder.bat`

Avoid:

- `MyScript.bat`
- `test1.bat`
- `stuff.bat`

## Folder Names

Use category-based names:

- `system`
- `network`
- `files`
- `dev`
- `installers`
- `templates`
