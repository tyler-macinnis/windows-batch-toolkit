# Script Template

Use this header format in every batch file:

```bat
@echo off
setlocal EnableExtensions

:: ============================================================
:: Script Name   : example-script.bat
:: Description   : Short description of the script.
:: Author        : Tyler MacInnis
:: Version       : 1.0.0
:: Created       : 2026-03-12
:: Updated       : 2026-03-12
:: Usage         : Explain how to run the script.
:: Requirements  : Windows CMD
:: Notes         : Add important warnings or behavior notes.
:: ============================================================
```
