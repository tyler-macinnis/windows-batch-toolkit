# Windows Batch Toolkit

A collection of useful Windows `.bat` scripts for administration, troubleshooting, development, file handling, and automation.

## Goals

- Keep scripts simple and practical
- Use a consistent structure and header format
- Organize scripts by category
- Make scripts easy to run from VS Code
- Support exporting all scripts into a single output folder

## Script Header Standard

Every batch file in this repository includes a standard header with:

- Script Name
- Description
- Author
- Version
- Created
- Updated
- Usage
- Requirements
- Notes

## Usage

Run a script directly:

```bat
scripts\system\show-system-info.bat
```

Or open the repo in VS Code and run a task.

## Contributing

When adding scripts:

1. Place the file in the correct subfolder
2. Use lowercase dash-separated names
3. Use the standard header format
4. Keep comments clear
5. Update the README if you add new categories
